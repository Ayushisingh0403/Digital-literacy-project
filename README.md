📌 Project Overview

This repository contains my Digital Literacy Portfolio, created as part of the CSE0001 – Digital Literacy course.

As a Student Digital Ambassador, this project focuses on helping students understand:
	•	Digital tools and platforms
	•	Professional online presence
	•	Safe and responsible internet usage
	•	Effective digital communication
	•	Cybersecurity awareness

The project is divided into five tasks, each aligned with a module of the course.
